# Phylogenomics of Apocynaceae (This repo is being updated over the weekend ... please be patient)
This project implements a state-of-the-art workflow to decode the evolutionary history of Dogbane family (Apocynaceae - Gentianales) using the Angiosperms353 universal probe set. This worflow assesses how gene signal strength - a concept pioneered by [Salichos & Rokas, 2013](https://www.nature.com/articles/nature12130) - scales in the phylogenomic era. We also provide here a path for integrative phylogenomic approach almost two decades after being proposed. [See](https://link.springer.com/article/10.1007/s10539-010-9218-2).

## Highlights
*   **Workflow:** From Illumina reads to species tree summary.
*   **Data:** Based on the Angiosperms353 probe set.
*   **Objective:** To determine the efficacy of _signal-bearing genes_ to illuminate processes underlying nodes in the phylogenetic tree at small taxonomic level (i.e. ancient and recent divergence, hybridisation and introgression).

For more details about this methodology, please visit the [tutorial](https://github.com/cassiabitencourt/Signal-based-phylogeny/wiki/workflow).
