# Phylogenomics of Apocynaceae
This project presents a comprehensive workflow to analyse the angiosperms353 data of Apocynaceae using cutting-edge bioinformatics tools. The aim is to build accurate phylogenetic trees, which can offer insights into the evolutionary history of this diverse family of flowering plants. Here we investigate in what extent gene signals of the angiosperms353 probe set may illuminate ancient and recent divergences at family level. The use of genes with strong signals have been highlighted as informative since Salichos & Rokas [2013](https://www.nature.com/articles/nature12130) - now we tested it in the phylogenomic era using an universal probe set.

For more details about this methodology, please visit the associated [methodology](https://github.com/cassiabitencourt/Phylogenomics-of-Apocynaceae/wiki).
