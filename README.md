# Phylogenomics of Apocynaceae
This project implements a state-of-the-art workflow to decode the evolutionary history of Dogbane family (Apocynaceae - Gentianales) using the Angiosperms353 universal probe set. By leveraging cutting-edge bioinformatics, we assess how gene signal strength - a concept pioneered by Salichos & Rokas [2013](https://www.nature.com/articles/nature12130) - scales in the phylogenomic era. 

## Highlights
*   **Workflow:** From Illumina reads to species tree summary.
*   **Data:** Based on the Angiosperms353 probe set.
*   **Objective:** To determine the efficacy of strong-signal genes in resolving ancient and recent divergence at family level.

For more details about this methodology, please visit the associated [methodology](https://github.com/cassiabitencourt/Phylogenomics-of-Apocynaceae/wiki).
